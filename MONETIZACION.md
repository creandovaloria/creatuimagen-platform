## Estrategia de lanzamiento

### Fase 1 — Validación (ahora)
- Regina como caso de uso real y portafolio
- Precio: cortesía / intercambio con Liz Barron
- Objetivo: tener URL funcionando para mostrar

### Fase 2 — Primeros clientes (mes 1-3)
- 3-5 eventos con Liz Barron como canal
- Precio Basic/Pro
- Iterar con feedback real

### Fase 3 — Escala (mes 4+)
- Stripe integrado para cobro automático
- Dashboard para que clientes suban sus propias fotos
- Liz como reseller con comisión automática
